# Outcomes of THA in Proximal Femoral Migration

R analysis and report for a longitudinal study evaluating the clinical and functional recovery of 24 patients 
with proximal femoral migration undergoing total hip arthroplasty.

## Contents

- Data Preprocessing: Cleaning and distribution analysis of clinical metrics, including normality assessment for continuous variables.
- Baseline Demographics: Summary of patient characteristics.
- Functional Recovery: Longitudinal analysis of Harris Hip Score (HHS) across four time points (Preoperative to 1 year)
using one-way repeated-measures ANOVA.
- Anatomical Correction: Statistical comparison of limb length discrepancy and flexion deformity using Wilcoxon signed-rank test.
- Coronal Realignment: Evaluation of pelvic obliquity improvements using McNemar's test.
- Publication-Ready Outputs: Pairwise comparison tables with Bonferroni correction and
longitudinal trajectory plots with 95% confidence intervals.
